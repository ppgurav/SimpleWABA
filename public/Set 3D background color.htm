ChatGPT said:

To apply a different background color or style (like a unique 3D color) only to messages on the right side, you'll need a way to determine whether the message is from the user or someone else.

✅ Step-by-step Solution

Assuming your message object has something like isOwnMessage or isUser to indicate it's from the user:

✅ Updated Code Example:

jsx
const TextMessage = ({ message }) => {
  const isOwnMessage = message.isOwnMessage; // adjust based on your data

  return (
    <div
      className={`${
        isOwnMessage
          ? "bg-gradient-to-br from-green-200 to-green-400 shadow-lg border border-green-300 self-end"
          : "bg-gradient-to-br from-blue-200 to-blue-400 shadow-lg border border-blue-300 self-start"
      } p-2 md:p-3 rounded-xl max-w-xs md:max-w-sm relative`}
    >
      <div
        dangerouslySetInnerHTML={{ __html: formatText(message.text, "text") }}
        className="whitespace-pre-wrap break-words"
      />
      {message.timestamp && (
        <div className="text-xs text-gray-500 mt-1 text-right">{message.timestamp}</div>
      )}
    </div>
  );
};

🔍 What's happening here:

  • isOwnMessage ? ... : ... — conditionally applies different background gradients based on the sender.

  • self-end for right alignment (own messages), self-start for left alignment (other messages).

  • Green gradient used for right-side/user messages, blue for others.

✅ Optional:

If you don't have isOwnMessage, you’ll need to determine it based on the message object (e.g. message.senderId === currentUserId).
